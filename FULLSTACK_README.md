# Full Stack Development Project

A complete full stack application with React frontend and Node.js/Express backend.

## Project Structure

```
├── frontend/          React application (Vite + React Router)
├── backend/           Node.js/Express API server
└── REQUIREMENTS.md    Technical requirements
```

## Quick Start

### Backend Setup
```bash
cd backend
npm install
cp .env.example .env
npm run dev
```

Backend runs on `http://localhost:5000`

### Frontend Setup
```bash
cd frontend
npm install
cp .env.example .env
npm run dev
```

Frontend runs on `http://localhost:3000`

## Tech Stack

### Backend
- Node.js
- Express.js
- MongoDB/Mongoose
- RESTful API
- CORS

### Frontend
- React 18
- Vite
- React Router
- Axios
- Responsive CSS

## Features

- RESTful API endpoints
- MongoDB/MySQL database integration
- Frontend-backend communication
- Responsive UI design
- Clean code architecture
- Git version control ready

## Development

Both frontend and backend support hot-reload during development. Run them in separate terminals.

## Deployment

- Backend: AWS, DigitalOcean, Heroku
- Frontend: Vercel, Netlify, AWS S3

See individual README files in `frontend/` and `backend/` directories for more details.
